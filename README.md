## Dependencies

- redux
- fontawesome
- material UI
- axios
- node-sass
- react testing library

## Setup instructions

Clone this repository 

```
git clone https://github.com/YoannMas/deliveroo-front-redux.git
```

Install dependencies

```
yarn install
```

Run it

```
yarn start
```
